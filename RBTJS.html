Red Black Tree demonstration

I present here a Javascript only demonstration of a Red Black Tree. You can call interactively the methods search(), insert(), remove() on the tree. The tree is initialized with 10 random values. To keep the display simple, the tree depth should not exceed 4. Therefore any insert that would result in a depth > 4 is not executed and an alert is displayed instead.
